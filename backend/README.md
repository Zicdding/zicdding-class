# zicdding-class-backend

직띵 클래스 웹 프로젝트 백엔드

## 시작하기

### 1. 터미널을 열고 backend로 이동합니다.

```
cd ./backend
```

### 2. yarn install

```
➜ yarn
yarn install v1.22.22
[1/5] 🔍  Validating package.json...
[2/5] 🔍  Resolving packages...
success Already up-to-date.
✨  Done in 0.15s.
```

### 3. yarn dev

```
➜ yarn dev
yarn run v1.22.22
$ nodemon --exec babel-node src/index.js
[nodemon] 3.1.1
```
